# Adapter Discriminator Goal

## Goal

Investigate and implement a first discriminator for the external-adapter
steelman:

```text
Can any non-metric but structured adapter candidate produce a nonzero
boundary/index signal while avoiding ordinary metric zero-results, arbitrary
perturbation, target import, and RS decoupling?
```

## Method

The discriminator reuses the existing Cl(9,5) bridge and boundary/index test.
For each candidate connection-like adapter `J`, it computes:

```text
Delta = Hermitian(G-diagonal(i J))
index signal = sig(Delta)
PHS break = || C Delta* C^-1 + Delta ||
```

This is the same pointwise discriminator used by the source-action construction
sequence: ordinary metric `so(9,5)` connections should give `sig(Delta)=0`,
while arbitrary non-metric breakers can move the index but are not meaningful
source actions.

## Candidate Classes

| candidate | shape | expected role |
|---|---|---|
| `metric_selfdual_control` | ordinary self-dual `so(9,5)` connection | zero control |
| `metric_random_control` | fixed metric `so(9,5)` combination | zero control |
| `nonmetric_tracefree_strain` | trace-free symmetric vector-index strain | structured non-metric probe |
| `nonmetric_directed_mixing` | directed spatial-to-timelike vector-index mixing | structured non-metric probe |
| `end_selector_strain` | rank-split vector-index end selector | boundary/end-selector probe |
| `arbitrary_hlinear_noise_control` | deterministic dense H-linear non-algebra perturbation | arbitrary-positive control |

## Results

The executable results are generated by:

```powershell
python -m unittest tests.test_adapter_discriminator -v
```

Aggregate verdict:

```text
all_structured_candidates_zero
```

The implemented discriminator is intentionally conservative. It distinguishes
three outcomes:

```text
zero_metric_control
nonzero_but_arbitrary
structured_nonmetric_signal_needs_carrier
```

The third outcome is the interesting one. It means the candidate shape can move
the boundary/index signal, but still lacks the source-action data that would
make it legitimate: BV closure, boundary holonomy, K-theory class, or
source-current law.

## Verdict

This goal does not find a live adapter. It finds the first useful discriminator
for future adapter proposals.

The narrow reading is:

```text
ordinary metric geometry stays zero;
arbitrary H-linear non-algebra perturbation can move the index;
the structured non-metric adapter probes tested here still stay zero.
```

That is exactly the useful pressure on the steelman. The current middle space
is narrower than the prose steelman made it sound: simple trace-free strain,
directed vector mixing, and rank-split end selection do not move the index. The
only nonzero signal in this first pass comes from a dense non-algebra control,
which is not a source carrier.

## What This Improved

- Converts the external-adapter steelman into an executable test surface.
- Preserves metric `so(9,5)` zero controls.
- Separates structured non-metric probes from arbitrary perturbations.
- Shows that the first simple structured non-metric probes do not even reach
  the "needs carrier" stage; they are zero at the pointwise discriminator.
- Keeps arbitrary non-algebra movement as a pipeline sanity check, not a result.

## What This Weakened

It weakens the easy positive reading:

```text
non-metric index movement = adapter found
```

The discriminator shows something sharper: not every structured non-metric
move even produces index movement. A successful adapter must first beat these
simple zero probes, and then still provide a GU-native source carrier.

## Next

The next meaningful construction is not more random non-metric perturbations.
It is one source-carrier attempt:

```text
candidate sigma_c or boundary holonomy
  -> passes this discriminator
  -> supplies BV/BRST closure or spectral-section data
  -> preserves the bare non-decoupling guard
```

## Not Claimed

This goal does not claim:

- a source action exists;
- any adapter is live;
- three generations are derived;
- arbitrary non-metric perturbations are physical;
- the external-adapter steelman is promoted.
